# OpenSea NFT Prompt Examples

## Portfolio And Discovery

```text
Show my NFTs on Ethereum.
Find trending NFT collections on OpenSea right now.
Show me the cheapest listings for this collection.
Summarize recent sales, owners, and floor-price trend for this collection.
```

## Collection Research

```text
Get OpenSea stats for this collection slug.
Show recent listing, sale, and transfer activity for this NFT collection.
Compare two NFT collections by floor price, volume, and owner count.
```

## Listings, Offers, And Buys

```text
Draft a listing for this NFT and show the exact order summary first.
Prepare an offer for this NFT on Base but stop before execution.
Show current listings for this token and tell me which order looks best.
Buy this NFT only after showing me the order hash, price, and protocol address.
Cancel my active NFT listing for this token.
```

## Transfers

```text
Prepare an NFT transfer to another wallet for review only.
Show me the exact NFT transfer summary before any wallet action.
Transfer this ERC1155 NFT and confirm the token, chain, and destination before execution.
```
